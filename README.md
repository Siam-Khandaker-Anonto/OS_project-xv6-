#OS_project-xv6

This is a basic os project based on xv6 where we have mainly implemented two features: "Symbolic links" and "Automatic file versioning". These features are not directly available in xv6. So by implementing these features, we can do these kinds of task in xv6 more fluently. Additionally,"Process info" as a system call has been added. All the test cases passes successfully so we can use these as a fully fledged feature inside the xv6 system.

To test the "Symbolic Links" feature, we have to give the command "symlinktest" in the xv6 commmand prompt window.
To test the "Automatic file versioning" feature, we have to give the command "versiontest" in the xv6 commmand prompt window.
